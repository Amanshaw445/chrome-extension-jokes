    <h1>Jokes Chrome Extension</h1>

    <h2>Description:</h2>
    <p>The Jokes Chrome Extension is a fun and light-hearted tool that provides users with a quick dose of humor right in their browser. With just a click of a button, users can fetch a new joke from an external API and enjoy a moment of laughter or amusement.</p>

    <h2>Features:</h2>
    <ul>
        <li><strong>Get New Joke Button:</strong> A prominent button on the extension popup interface allows users to easily fetch a new joke.</li>
        <li><strong>Dynamic Joke Display:</strong> Jokes fetched from the API are dynamically displayed on the extension popup, providing users with either single-line or two-part jokes.</li>
        <li><strong>Error Handling:</strong> The extension includes error handling to ensure smooth operation even in case of network issues or API failures.</li>
    </ul>

    <h2>Installation:</h2>
    <ol>
        <li><strong>Download:</strong> Download the extension files from the repository.</li>
        <li><strong>Install Extension:</strong> Open Google Chrome and navigate to <code>chrome://extensions/</code>. Enable Developer Mode by toggling the switch on the top right.</li>
        <li><strong>Load Extension:</strong> Click on "Load unpacked" and select the folder containing the extension files.</li>
        <li><strong>Enjoy:</strong> Once installed, click on the extension icon to view jokes and have a laugh!</li>
    </ol>

    <h2>Usage:</h2>
    <ul>
        <li>Upon installation, click on the extension icon in the Chrome toolbar to open the popup.</li>
        <li>The popup displays a joke fetched from the external API. If the joke is in two parts, the setup and delivery are shown together.</li>
        <li>Click on the "<strong>Get New Joke</strong>" button to fetch a fresh joke and enjoy endless laughter.</li>
    </ul>
